# NLP-Comp-Master
nlp 比赛代码模版

## 文件结构
--src 源代码  
    |--net 总体模型文件
    |--layers 公共网络模块
    |--engine 训练、验证、推断模块
    |--metrics 指标模块
    |--tools 训练与测试入口
    |--config 实验配置模块
    |--utils 工具类模块
    |--preprocessing 预处理模块
--log 实验日志
--cache 模型保存
--submit 提交文件夹
--raw_data 训练、测试、数据
    |--train
    |--test
--run.sh 运行脚本
